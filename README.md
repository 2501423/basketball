# 🏀 Houston Rockets 試合結果管理サイト

Houston Rockets の試合結果を記録・可視化できる個人開発のWebアプリです。

## 概要

「今のチームの状態を直感的に知りたい」という自分自身の課題から制作しました。
数字だけでなく、色・グラフ・一覧で試合データを多角的に確認できます。

## 使用技術

- HTML / CSS / JavaScript
- Chart.js（得点推移グラフ）
- localStorage（データ永続化）

## 主な機能

- 試合結果の入力・編集・削除
- 勝率のリアルタイム計算・表示
- 勝敗に応じたカードの色分けUI（勝ち：青 / 負け：赤）
- 得点推移の折れ線グラフ表示
- 最多得点選手の記録・一覧表示
- 相手チームのロゴ表示（NBA全30チーム対応）
- localStorage によるデータ保存（ページ更新後も保持）
- レスポンシブ対応（スマートフォン対応）
